# Operational Analytics & Business Intelligence Portfolio

This repository contains operational models, automated billing systems, and business intelligence dashboards designed to streamline supply chain logistics and financial tracking.

## 📁 Project Directory

### 1. Warehouse Management System (WMS)
* **Status:** Dashboard preview available (Full `.xlsm` available upon request)
* **Description:** Automated inventory and stock allocation tracking system using complex Excel logic to manage supply chain constraints and distribution routing.

### 2. Automated Billing & Invoice System
* **Status:** Template preview available 
* **Description:** VBA-engineered automated invoice generation system designed to eliminate manual data entry and accelerate the billing cycle.

### 3. Dynamic Investment Fund Tracker
* **Status:** Dashboard preview available
* **Description:** Built a multi-currency (NGN/USD) dashboard using VBA and Advanced Excel, featuring real-time portfolio tracking and dynamic data validation to automate performance analysis for stakeholders.

### 4. COVID Vaccine Tracker
* **Link:** https://public.tableau.com/shared/B4R68Z6M3?:display_count=n&:origin=viz_share_link
* **Description:** Interactive dashboard visualizing key performance indicators (KPIs) for executive decision-making.
